# fatality.win CS2 — FAQ

**Q: How do I switch between RageBot and LegitBot?**  
A: Menu → Aimbot → Mode → select Rage or Legit. You can bind a hotkey to toggle between modes mid-match: Aimbot → Mode Toggle Key.

**Q: The resolver isn't hitting players with anti-aim.**  
A: Resolver performance depends on having enough information about the enemy's movement. Adjust Menu → Aimbot → Resolver → Mode to "Auto" and let it adapt. For specific anti-aim patterns, load a community resolver script that handles them.

**Q: XQZ chams are invisible through walls.**  
A: Enable them: Menu → Visuals → Players → Chams → Material: XQZ → Occluded Color: set to any visible color. Both Visible and Occluded colors need to be set.

**Q: My config disappeared after reinstalling.**  
A: Configs saved with cloud sync are recoverable — log into your dashboard at fatality.win and re-sync. Local-only configs (sync disabled) are not recoverable without a manual backup.

**Q: What's the difference between Fakelag and regular lag?**  
A: Fakelag holds your outgoing packets for a set number of ticks then releases them in a burst, creating a desync effect that makes your real position harder to predict for enemies. It's configured under Misc → Fakelag.

**Q: Can I use my own knife/skin without owning it?**  
A: Yes — Fatality's inventory changer overrides what your client renders locally. Other players see your actual inventory. The skins are visual-only on your end.
